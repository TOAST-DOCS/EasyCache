## EasyCache 릴리스 노트
**Database > EasyCache > 릴리스 노트**

### 2026. 03. 24.

#### 기능 개선/변경
- Valkey 8.1.6을 지원합니다.
- 파라미터 그룹과 노드의 redis.conf 설정 차이를 콘솔에서 확인하고, 동기화할 수 있는 기능을 추가했습니다.

### 2026. 01. 27.

#### 기능 개선/변경
- 실제 장애 조치 수행 완료 후 읽기 전용 도메인의 IP 갱신을 위한 버튼을 콘솔에 노출하도록 변경했습니다.
- ID로도 캐시 리스트를 검색할 수 있도록 변경했습니다.
- 사용자 가이드에 포함된 일부 이벤트 목록을 현행화했습니다.
- Valkey 8.x 이상 버전들의 일부 파라미터 이름을 최신 버전으로 변경했습니다.

#### 버그 수정
- 데이터 가져오기에 사용할 RDB 파일명 중 특정 문자열이 포함되었을 경우 오류가 발생하는 버그를 수정했습니다.
- 이벤트 화면에서 다운로드 가능한 CSV 파일 내 헤더 항목의 다국어 표시를 수정했습니다.

### 2025. 11. 15.

#### 기능 개선/변경

- Valkey 8.1.4를 지원합니다.
- 일부 파라미터에 대한 가이드 문구를 개선했습니다.
- 이벤트 구독 > 이벤트 코드 목록을 '시작', '완료', '실패' 순으로 정렬되도록 개선했습니다.

#### 버그 수정

- 특정 조건에서 엔진 버전 업그레이드 시 실패하는 버그를 수정했습니다.
- 캐시, 파라미터 그룹 작업 중일 때 수정 버튼이 활성화되는 버그를 수정했습니다.
- 노드 삭제 중 고가용성 재설정 버튼이 노출되는 버그를 수정했습니다.
- 다른 리전 노드의 모니터링 탭/사용자 탭에서 오류 메시지가 출력되는 버그를 수정했습니다.
- 서버 대시보드 탭에서 다른 리전 노드가 조회되는 버그를 수정했습니다.

### 2025. 09. 23.

#### 기능 개선/변경
- 콘솔 화면을 개편했습니다.
- 기존 기능을 변경 및 개선했습니다.

  | 변경 전 | 변경 후 |

### 2025. 07. 29.

#### 버그 수정

- 하이퍼바이저 점검으로 인한 EasyCache 재시작 관련 이메일 발송이 누락되는 버그를 수정했습니다.

#### 버그 수정

- 레플리케이션 이용 불가 상태에서 레플리케이션을 삭제해도 복제 그룹 상태가 정상으로 표시되지 않는 문제를 수정했습니다.
- 복제 그룹 복원에서 복제 그룹이 삭제된 백업을 복원할 때, 선택 가능한 복제 그룹이 표시되지 않는 문제를 수정했습니다.
