
## Database > EasyCache > 릴리스 노트

### 2020. 01. 21.

#### 기능 개선

- 콘솔 화면과 이벤트 메세지의 일본어 대응

### 2019.12.24

#### 신규 상품 출시

- TOAST EasyCache는 Redis(REmote DIctionary Server)를 클라우드 환경에서 제공하는 상품입니다.
- 간단한 설정으로 고가용성(자동HA)의 Redis 서버를 사용할 수 있습니다.
- Redis 3.2.12 버전을 제공합니다.
